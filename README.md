# MyNotesApp

**وصف المشروع:**  
تطبيق الملاحظات هو تطبيق لتسجيل الملاحظات السريعة، يسمح للمستخدم اضافة وحذف وعرض الملاحظات من خلال واجهة بسيطة ب استخدام php و json
## التكنولوجيا المستخدمة
- PHP
- JavaScript
- CSS
- JSON (لتخزين الملاحظات)
- Docker

## تشغيل المشروع باستخدام Docker
1. فتح نافذة الأوامر cmd :
انتقل الى المسار الرئيسي للمشروع عبر نافذة الأوامر

2. بناء صورة Docker للمشروع :
اكتب الأمر التالي ثم اضغط على زر Enter:
docker build --tag projhw .

3. تشغيل الحاوية (Container) من الصورة المُنشأة :
بعد اكتمال البناء، اكتب الأمر التالي ثم اضغط على زر Enter:
docker run --name projhimg -p 5000:80 projhw

4. الوصول إلى التطبيق عبر المتصفح:
افتح متصفح الويب واكتب العنوان التالي في شريط العناوين:
http://localhost:5000/

## ايقاف المشروع
1. فتح تطبيق Docker Desktop
2. ايقاف الحاوية الخاصة بالمشروع من خلال زر الايقاف
